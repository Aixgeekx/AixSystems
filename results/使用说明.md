# AixSystems · 使用说明

> 离线本地时间管理系统。所有可双击脚本统一放在 `results/` 目录。

## 启动入口

| 文件 | 用途 | 说明 |
|---|---|---|
| `桌面版.bat` | Electron 桌面版 | 日常使用推荐 |
| `启动.bat` | 浏览器生产模式 | 本地预览生产构建 |
| `开发.bat` | 浏览器开发模式 | 支持热更新 |
| `桌面版-开发.bat` | Electron 开发模式 | 支持热更新 |
| `打包.bat` | 生成 Windows 安装包 | 输出到 `desktop/dist-installer/` |
| `打包-便携版.bat` | 生成便携压缩包 | 输出到 `results/portable/` |
| `创建桌面快捷方式.bat` | 创建桌面快捷方式 | 会生成 3 个快捷方式 |
| `_诊断.bat` | 环境诊断 | 检查 Node/npm/关键文件 |

## 使用方法

1. 打开仓库根目录下的 `results/`
2. 日常使用双击 `results/桌面版.bat`
3. 修改源码时双击 `results/开发.bat` 或 `results/桌面版-开发.bat`
4. 分发安装包时双击 `results/打包.bat`
5. 需要“解压即用”版本时双击 `results/打包-便携版.bat`

## 首次启动

- 浏览器版会自动安装 `code/` 依赖并构建前端产物
- 桌面版会额外安装 `desktop/` 依赖
- 所有脚本都按相对路径回到仓库根目录执行,所以必须保留 `code/`、`desktop/`、`data/` 目录结构

## 目录结构

```text
AixSystems/
├─ code/                       应用源码
├─ desktop/                    Electron 桌面壳
├─ data/                       本地备份目录
├─ Aix_tools/                  辅助脚本
├─ results/
│  ├─ 桌面版.bat
│  ├─ 启动.bat
│  ├─ 开发.bat
│  ├─ 桌面版-开发.bat
│  ├─ 打包.bat
│  ├─ 打包-便携版.bat
│  ├─ 创建桌面快捷方式.bat
│  ├─ 创建桌面快捷方式.ps1
│  ├─ _诊断.bat
│  └─ 使用说明.md
└─ README.md
```

## 便携分发说明

- Windows 安装包产物在 `desktop/dist-installer/`
- 便携压缩包产物在 `results/portable/AixSystems-Windows-Portable.zip`
- 该压缩包内包含便携版 `.exe`、`LICENSE` 和本说明，解压后即可直接运行
- 便携版会把 `userData/` 放在程序同目录，方便把整个文件夹直接复制到其他 Windows 电脑继续使用

## 小组件主题

- 浮动小组件支持 6 种主题: 白天、黑夜、简约、赛博朋克、渐变、复古
- 小组件主题在浮动窗右上角设置面板中切换
- 主题切换会直接改变小组件配色和质感，不再固定为白色

## 常见问题

- 桌面版打不开: 先运行 `results/_诊断.bat`
- 浏览器版端口占用: 检查 `5173` 或 `4173` 端口是否被其他程序占用
- 打包慢: Electron 首次下载体积较大,等待时间会明显更长
