# AixSystems · 使用说明

> 离线本地时间管理系统。所有可双击脚本统一放在 `results/` 目录。

当前发布收口版本: `v0.23.4`

## 启动入口

| 文件 | 用途 | 说明 |
|---|---|---|
| `桌面版.bat` | Electron 桌面版 | 日常使用推荐 |
| `启动.bat` | 浏览器生产模式 | 本地预览生产构建 |
| `开发.bat` | 浏览器开发模式 | 支持热更新 |
| `桌面版-开发.bat` | Electron 开发模式 | 支持热更新 |
| `打包.bat` | 生成 Windows 安装包 | 输出到 `desktop/dist-installer/` |
| `打包-便携版.bat` | 生成目录式便携压缩包 | 输出到 `results/portable/` |
| `打包-单文件便携版.bat` | 生成单文件 portable.exe | 输出到 `desktop/dist-installer/` |
| `创建桌面快捷方式.bat` | 创建桌面快捷方式 | 会生成 3 个快捷方式 |
| `_诊断.bat` | 环境诊断 | 检查 Node/npm/关键文件 |

## 使用方法

1. 打开仓库根目录下的 `results/`
2. 日常使用双击 `results/桌面版.bat`
3. 修改源码时双击 `results/开发.bat` 或 `results/桌面版-开发.bat`
4. 分发安装包时双击 `results/打包.bat`
5. 需要“解压即用”目录版时双击 `results/打包-便携版.bat`
6. 需要单文件 portable.exe 时双击 `results/打包-单文件便携版.bat`

## 首次启动

- 浏览器版会自动安装 `code/` 依赖并构建前端产物
- 桌面版会额外安装 `desktop/` 依赖
- 所有脚本都按相对路径回到仓库根目录执行,所以必须保留 `code/`、`desktop/`、`data/` 目录结构

```text
│  ├─ 桌面版.bat

│  ├─ 桌面版-开发.bat

│  ├─ 打包-单文件便携版.bat

```

## 便携分发说明

- Windows 安装包产物在 `desktop/dist-installer/`
- 当前版本安装包命名为 `AixSystems-0.23.4-Setup.exe`
- 目录式便携压缩包产物在 `results/portable/AixSystems-Windows-Portable.zip`
- 该压缩包内包含展开后的 `AixSystems.exe`、`resources/`、`locales/`、`LICENSE`、`启动AixSystems.bat` 和本说明，解压后即可直接运行
- 目录式便携版会通过 `AixSystems.portable` 标记把 `userData/` 固定在程序同目录，方便把整个文件夹直接复制到其他 Windows 电脑继续使用
- 如需单文件便携分发，可额外使用 `results/打包-单文件便携版.bat`
- 如果启动后出现白屏，请先确认使用的是 `v0.19.1` 及以上修复版本
- 如果首页在深色主题下仍出现发白卡片或淡字，请确认使用的是 `v0.19.2` 及以上版本
- 如果需要成长仪表盘、成就徽章、周期复盘和 27 款主题，请使用 `v0.21.9` 及以上版本
- 如果需要记忆曲线真实复习提醒队列，请使用 `v0.22.0` 及以上版本
- 如果需要首页紧凑应用入口，请使用 `v0.22.1` 及以上版本
- 如果需要首页紧凑模式下的搜索、分组筛选和收藏置顶，请使用 `v0.22.2` 及以上版本
- 如果需要复习中心查看今日待复习和未来复习节点，请使用 `v0.22.3` 及以上版本
- 如果需要专注页月同比分析，请使用 `v0.22.4` 及以上版本
- 如果需要成长仪表盘 Markdown 报告导出，请使用 `v0.22.5` 及以上版本
- 如果需要复习完成态追踪和掌握率统计，请使用 `v0.22.6` 及以上版本
- 如果需要“需巩固”自动回流复习，请使用 `v0.22.7` 及以上版本
- 如果需要配置“需巩固”后的复习强度和回流间隔，请使用 `v0.22.8` 及以上版本
- 如果需要首页紧凑应用入口拖拽排序，请使用 `v0.22.9` 及以上版本
- 如果需要选择性导出、Manifest 备份清单和数据主权评分，请使用 `v0.23.0` 及以上版本

## 小组件主题

- 浮动小组件支持 7 种模式: 跟随全局、白天、黑夜、简约、赛博朋克、渐变、复古
- 小组件主题在浮动窗右上角设置面板中切换，跟随全局时会自动同步当前主界面主题
- 主题切换会直接改变小组件配色和质感，赛博朋克等深色风格下不再出现白色设置面板

## 常见问题

- 桌面版打不开: 先运行 `results/_诊断.bat`
- 浏览器版端口占用: 检查 `5173` 或 `4173` 端口是否被其他程序占用
- 打包慢: Electron 首次下载体积较大,等待时间会明显更长
