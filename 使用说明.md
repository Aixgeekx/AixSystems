# AixSystems · 使用说明

> 离线本地时间管理系统 — 数据全部存在本机,零服务器依赖。

## 五种启动方式(双击即用)

| 文件 | 用途 | 启动速度 |
|---|---|---|
| **桌面版.bat** | 独立窗口桌面应用 (Electron),首次使用需安装 ~180MB | 首次约 1 分钟,之后秒开 |
| **启动.bat** | 浏览器打开 (生产模式,已优化压缩) | 首次约 30 秒,之后秒开 |
| **开发.bat** | 浏览器打开 + **代码热更新**,修改 `code/src/` 自动刷新 | 5 秒 |
| **桌面版-开发.bat** | Electron 窗口 + 代码热更新 | 10 秒 |
| **打包.bat** | 打包成 `.exe` Windows 安装包,可分发 | 首次 5-10 分钟 |
| **创建桌面快捷方式.bat** | 一次性运行,在桌面生成 3 个快捷方式 | 1 秒 |

## 推荐用法

- **日常使用** → 桌面上双击「**AixSystems(桌面版)**」快捷方式
- **修改代码** → 双击「开发.bat」或「桌面版-开发.bat」,改 `code/src/` 下任意文件自动生效
- **分发给别人** → 双击「打包.bat」,生成 .exe 安装包
- **手机/其他电脑访问** → 启动「启动.bat」后,同 Wi-Fi 里用浏览器访问本机 IP:4173

## 首次使用

1. **一次性准备**:双击「创建桌面快捷方式.bat」,桌面会多 3 个快捷方式
2. **启动**:双击「AixSystems(桌面版)」
3. 首次启动会自动:
   - 安装 npm 依赖 (约 30 秒)
   - 构建生产包 (约 20 秒)
   - 安装 Electron (约 1 分钟,仅桌面版)
4. 之后每次启动都是秒开。

## 修改代码

1. 双击「开发.bat」启动开发服务器,浏览器自动打开
2. 用任意编辑器 (VS Code 推荐) 打开 `code/src/`
3. 修改 `.tsx` / `.ts` 文件保存,**浏览器自动刷新**,IndexedDB 数据不丢失
4. 想验证生产构建效果:`Ctrl+C` 停开发服务器,再双击「启动.bat」

### 常见修改入口

| 想改什么 | 去哪里 |
|---|---|
| 品牌名 / 文案 / DB 名 | `code/src/config/constants.ts` |
| 左侧菜单项 | `code/src/config/routes.ts` |
| 17 种事项类型 | `code/src/config/itemTypes.ts` |
| 主题壁纸颜色 | `code/src/config/themes.ts` |
| 内置节日 | `code/src/config/festivals.ts` |
| 某个页面布局 | `code/src/pages/{对应模块}/` |
| 整体视觉风格 | `code/src/components/Layout/` |
| 窗口大小/标题 | `desktop/main.cjs` |
| 应用图标 | `Aix_tools/gen_icon.cjs` 改脚本后重新运行 |

## 数据备份(重要)

**所有数据存在 IndexedDB**,清浏览器数据或重装 Windows 会全部丢失。请定期通过:

左侧菜单 → 设置 → 导入导出 → 「保存备份到本地」

- **桌面版**: 备份直接写入 `data/` 目录(打包后写 `%APPDATA%/AixSystems/data/`)
- **浏览器版**: 触发浏览器下载到下载文件夹

换机/换浏览器时,同一页面「选择 JSON 文件」恢复。

> ⚠️ **桌面版与浏览器版数据不互通** — 两边的 IndexedDB 独立。若要切换,先在一边导出 JSON,再到另一边导入。

## 桌面版 VS 浏览器版的差异

| 能力 | 浏览器版 | 桌面版 (Electron) |
|---|---|---|
| 数据存储 | 浏览器 IndexedDB | Electron 独立 IndexedDB |
| 备份保存位置 | 触发浏览器下载 | **直接写入 `data/` 目录** |
| 导入文件 | 上传表单 | **系统原生文件选择器** |
| Service Worker 离线 | ✓ 支持 | ✗ 不需要 (本地直接加载) |
| 通知 | 浏览器通知 | 系统原生通知 |
| 打开外链 | 同标签或新标签 | 系统默认浏览器 |
| 可分发 | 部署到 HTTP 服务 | **.exe 安装包 / 便携版** |

## 打包为 .exe 安装包

**方式 1:一键脚本(推荐)**
双击 `打包.bat`,自动构建 + 打包,输出到 `desktop/dist-installer/`:
- `AixSystems-0.1.0-Setup.exe` — NSIS 安装包(约 80MB,有向导)
- 或 `npm run dist:portable` 生成绿色便携版(约 80MB,单文件直接运行)

**方式 2:手动命令**
```bash
cd code && npm run build          # 构建 dist
cd ../desktop && npm install      # 首次
npm run dist                      # 生成 NSIS 安装包
# 或 npm run dist:portable        # 生成便携版
```

## 故障排除

- **桌面版打不开 / 白屏**: 检查 `code/dist/index.html` 是否存在。不存在的话 `cd code && npm run build`
- **浏览器版打不开**: 确认 4173 端口没被占用,`netstat -ano | findstr :4173` 查占用进程
- **首次安装很慢**: 用淘宝 npm 镜像加速,`npm config set registry https://registry.npmmirror.com` 后重新双击
- **Electron 下载慢 / 超时**: 先设环境变量 `set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/` 再双击
- **通知不弹**: 浏览器右上角点「允许通知」,或设置→系统设置→「开启浏览器通知」
- **应用锁忘密码**: 桌面版删 `%APPDATA%/AixSystems/` 下 IndexedDB;浏览器版 F12→Application→IndexedDB→aixsystems_db→settings 删 `appLockPasswordHash`

## 目录结构

```
AixSystems/
├─ 启动.bat                  双击 - 浏览器 + 生产模式
├─ 开发.bat                  双击 - 浏览器 + 热更新
├─ 桌面版.bat                双击 - Electron 独立窗口
├─ 桌面版-开发.bat           双击 - Electron + 热更新
├─ 打包.bat                  双击 - 生成 .exe 安装包
├─ 创建桌面快捷方式.bat      双击 - 生成 3 个桌面快捷方式
├─ 使用说明.md               本文件
├─ code/                     源代码 (Vite + React + TS)
│  ├─ src/                   应用代码
│  ├─ dist/                  构建产物 (首次启动后生成)
│  └─ node_modules/          依赖 (首次启动后生成)
├─ desktop/                  Electron 桌面壳
│  ├─ main.cjs               主进程 (加载窗口 + IPC)
│  ├─ preload.cjs            渲染进程 API 桥 (window.sgx)
│  ├─ package.json           electron-builder 配置
│  ├─ build/icon.ico         应用图标 (小写 x,自动生成)
│  ├─ dist-installer/        打包产物 (打包后生成)
│  └─ node_modules/          Electron (首次启动桌面版后生成)
├─ data/                     JSON 备份目录(桌面版自动写入)
├─ title/                    调研资料
├─ Aix_tools/                辅助脚本 (含图标生成器)
└─ results/                  最终交付物
```
